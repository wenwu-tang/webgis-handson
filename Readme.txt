****** README File for the dataset ******
Raster datasets for USDA Forest Service Santee Experimental Forest

The dataset includes 9 rasters in geotiff formats that cover the Santee Experimental Forest. 
The data source of these geotiff raster datasets is USDA NAIP Public Image Service (USDA: US Department of Agriculture; NAIP: National Agriculture Imagery Program; https://naip-usdaonline.hub.arcgis.com/).
The year of the data collection: 2023
